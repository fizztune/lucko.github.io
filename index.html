My projects

An abridged list of my publicly available projects.

LuckPerms

  • LuckPerms is a permission plugin for Minecraft servers, written in Java.
  • At time of writing, it's been downloaded ~1.5M times, has ~250,000 monthly active users, and a Discord community of ~10,000 people.
  • More information can be found on the project homepage, luckperms.net.

spark

  • spark is a performance monitoring plugin with a built-in profiler for Minecraft servers, written in Java and React.
  • More information can be found on the spark and spark-viewer GitHub repositories.

helper

  • helper is a Bukkit plugin development toolkit. Designed for plugin developers, it provides a number of utilities and extended APIs to support rapid and easy development of plugins.
  • More information can be found on the helper GitHub repository.
